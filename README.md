# Resume Screening System

## 📌 Project Overview
The Resume Screening System is a Machine Learning and NLP-based project developed to automatically rank and screen resumes according to a given job description.

This system compares candidate resumes with required job skills and calculates similarity scores to identify the most suitable candidates.

---

## 🎯 Objective
To build an automated resume screening system that:
- Analyzes resume content
- Matches resumes with job descriptions
- Ranks candidates based on skill similarity
- Helps simplify the recruitment process

---

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (Natural Language Processing)
